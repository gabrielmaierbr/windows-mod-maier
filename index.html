Windows MOD Lite

Windows modificado a partir de NTLite, removendo recursos desnecessários (Bloatwares) instalados de fábrica pela Microsoft.
Não está incluso nenhuma forma de ativação ilegal, nenhum Aplicativo/Programa necessário para funcionamento do sistema a nível crítico foi removido.
